# 简介
一个基于unity3d的开发框架。
框架力致于打造一个简单、高效的开发流，更注重于基于数据驱动的工具开发（工具才是第一生产力，框架只是辅助而已）。
<br>有任何问题 直接提issue，1天内必解决

## 框架特点
  <br>**不开发工具的框架都是耍流氓!!!!**<br>
  <br>所有方案制作，都致力于开发者0学习成本，跟普通开发一样舒爽。
  <br>1.业务逻辑自动注册
  <br>2.热更代码跟非热更开发一样舒爽
  <br>3.ui工作流代码自动生成，数据自动绑定等
